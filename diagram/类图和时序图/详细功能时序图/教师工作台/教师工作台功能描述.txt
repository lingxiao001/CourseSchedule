教师工作台功能模块完整描述
==============================

模块概述
--------
教师工作台是专为教师用户设计的综合性管理平台，提供了课程管理、教学班管理、课表查看、
教学资源管理等核心功能。该平台旨在提高教师的工作效率，简化教学管理流程，为教师提供
便捷的教学支持工具。

核心功能模块
-----------

1. 按周查看课表功能
   功能描述：教师可以按周查看个人课表，支持周次切换、课程详情查看、课表导出等功能。
   主要特性：
   - 周次灵活切换，支持查看任意周的课表安排
   - 课程详情展示，包含课程信息、学生名单、出勤情况
   - 多格式导出支持（PDF、Excel）
   - 课表统计分析，包括课时统计、时间分布分析
   - 上课提醒设置，支持邮件、短信、系统通知
   - 响应式设计，适配PC和移动端

2. 教学班管理功能
   功能描述：教师可以创建、编辑、删除个人的教学班，管理教学班的基本信息和学生容量。
   主要特性：
   - 教学班信息管理：代码、名称、容量、当前人数
   - 关联课程选择，支持从个人课程中选择
   - 学生人数实时统计和容量控制
   - 教学班代码唯一性验证
   - 批量操作和搜索功能
   - 数据完整性检查，防止误删有学生选课的教学班

3. 课程管理功能
   功能描述：教师可以管理个人负责的课程信息，包括课程创建、编辑、删除等操作。
   主要特性：
   - 课程基本信息管理：代码、名称、学分、课时、描述
   - 课程代码唯一性验证
   - 关联性检查，避免删除有教学班的课程
   - 搜索和筛选功能
   - 分页显示和批量操作
   - 课程统计和分析功能

4. 我的课程查看功能
   功能描述：教师可以查看个人教授的所有课程概览，包括课程统计和详细信息。
   主要特性：
   - 课程概览展示，包含课程基本信息和教学班统计
   - 课程详情查看，支持深入了解课程详细信息
   - 教学班快速跳转，便于管理相关教学班
   - 课程信息导出功能
   - 教学任务统计和分析

详细功能流程
-----------

按周查看课表流程：
1. 教师身份验证和权限检查
2. 查询教师的教学班信息
3. 获取当前周的课表安排
4. 查询上课教室和相关信息
5. 组装并显示课表数据
6. 支持周次切换和课程详情查看
7. 提供导出和统计功能

教学班管理流程：
1. 查询教师的教学班列表
2. 支持添加新教学班操作
   - 选择关联课程
   - 填写教学班信息
   - 验证代码唯一性
   - 保存教学班数据
3. 支持编辑现有教学班
   - 修改教学班信息
   - 更新容量和其他属性
4. 支持删除教学班
   - 检查是否有学生选课
   - 安全删除教学班数据

课程管理流程：
1. 查询教师的课程列表
2. 支持添加新课程操作
   - 填写课程基本信息
   - 验证课程代码唯一性
   - 保存课程数据
3. 支持编辑现有课程
   - 修改课程信息
   - 更新学分、课时等属性
4. 支持删除课程
   - 检查是否有关联教学班
   - 安全删除课程数据

我的课程查看流程：
1. 查询教师的教学班数据
2. 获取关联的课程信息
3. 统计每个课程的教学班数量
4. 展示课程概览列表
5. 支持查看课程详情
6. 提供管理功能快速入口

业务规则和约束
-----------

数据完整性约束：
- 教学班代码在系统中必须唯一
- 课程代码在系统中必须唯一
- 教学班必须关联到有效的课程
- 课程删除前必须确保无关联教学班

权限和安全约束：
- 教师只能管理自己的课程和教学班
- 所有操作都需要通过身份验证
- 敏感操作需要二次确认
- 操作日志记录和审计

业务逻辑约束：
- 教学班容量不能小于当前选课人数
- 有学生选课的教学班不能删除
- 有关联教学班的课程不能删除
- 课表时间不能冲突
