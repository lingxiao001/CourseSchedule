## 🛠️ 技术架构

### 后端技术栈
- **框架**：Spring Boot + Spring Security
- **数据库**：MySQL + JPA/Hibernate  
- **认证**：JWT Token
- **API**：RESTful Web Services
- **事务管理**：声明式事务

### 前端交互
- **页面渲染**：Vue.js组件化
- **状态管理**：Pinia/Vuex
- **HTTP请求**：Axios + 拦截器
- **UI组件**：Element Plus

### 安全机制
- **认证方式**：JWT Bearer Token
- **权限控制**：基于角色的访问控制(RBAC)
- **数据验证**：前后端双重验证
- **安全传输**：HTTPS协议
